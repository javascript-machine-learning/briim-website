## Setup

* `git clone git@github.com:rwieruch/gatsby-movement.git`
* `cd gatsby-movement`
* `npm install`
* `gatsby develop`
* visit http://localhost:8000/

## Setup via Gatsby CLI

* `gatsby new gatsby-movement git@github.com:rwieruch/gatsby-movement.git`
* `cd gatsby-movement`
* `npm install`
* `gatsby develop`
* visit http://localhost:8000/
